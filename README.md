üìò Project Overview

This project demonstrates the implementation of Automation Testing using Selenium WebDriver, Java, and TestNG in Eclipse IDE. 
The main objective of this project is to automate functional workflows of a sample web application to ensure the reliability and efficiency of its core features. 
The scripts are designed to perform automated actions such as launching browsers, creating new user accounts, handling forms, managing radio buttons, entering data, and verifying successful login and UI interactions.
The project provides a comprehensive example of how manual test cases can be translated into automated scripts using Selenium, reducing testing time and improving consistency.
It follows the Page Object Model (POM) structure to make the code modular, reusable, and easy to maintain.
